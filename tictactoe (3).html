🔥 הגדרת Firebase – פעם אחת בלבד

כדי שמשחק הרשת יעבוד בין מכשירים שונים דרך האינטרנט, צריך Firebase (חינמי לחלוטין):

1. היכנס ל-console.firebase.google.com עם חשבון Google
2. לחץ Add project ← שם לפרויקט ← Continue (השאר הכל כברירת מחדל)
3. בפרויקט: Build ← Realtime Database ← Create database ← בחר "Start in test mode" ← Enable
4. לחץ על סמל ה-</> (Web) בדף הראשי ← Register app ← העתק את firebaseConfig
5. הדבק את הקוד כאן ↓ (כולל הסוגריים {})
